# Bunstone Starter

A minimal starter project for building applications with the Bunstone framework.

## Getting Started

1. **Install dependencies:**

   ```bash
   bun install
   ```

2. **Run in development mode:**

   ```bash
   bun dev
   ```

3. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Project Structure

- `src/main.ts`: Entry point for the application.
- `src/app.module.ts`: Root module definition.
- `src/controllers/`: Application controllers.
- `src/services/`: Application services (providers).
- `src/views/`: React components for SSR rendering.
